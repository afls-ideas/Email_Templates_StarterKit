# 03 - Mechanism of Action

## Description
Explains how Osteovance works at the molecular level using a step-by-step numbered layout. Designed for a clean, scientific presentation of a drug's mechanism of action for osteoporosis treatment.

## Primary Color
Royal Blue (#2E5090)

## LSC Features Used
- **LSC_PresentationLinks** (`moaDeckLink`) - Link to a detailed MOA slide deck for deeper engagement
- **Mustache Variables** - `{{recipient.name}}`, `{{sender.firstname}}`, `{{sender.lastname}}`, `{{sender.title}}`, `{{sender.company}}`

## Layout Features
- Centered header with scientific subtitle and product name
- Four numbered circular step indicators with dashed connector lines
- Progressive color lightening across steps (dark to light blue)
- Presentation link card with icon

## Customization Notes
- Replace the four MOA steps with your product's specific mechanism
- Adjust the number of steps (add or remove step rows and connectors)
- Update the presentation link section with your actual CLM deck reference
- Modify biomarker and trial data references in step descriptions
