# 13 - Clinical Guidelines Update

## Description
Communicates updated clinical treatment guidelines (ACC/AHA heart failure) to HCPs, highlighting key changes and how Cardivex fits into new recommendations. Features color-coded change badges and recommendation tier visuals.

## LSC Features Used
- **LSC_RichTextInput** (`relevance_note`) - Rep adds a personalized note on why the guideline changes matter for this specific HCP's practice

## Template Sections
- Guideline organization badge bar (ACC/AHA)
- Header with guideline title and summary
- Color-coded change cards (NEW / UPDATED / EXPANDED)
- Recommendation strength tier visualization (Class I, IIa, IIb)
- Personalized relevance note
- CTA to full guidelines summary

## Color Palette
- Primary: `#0E6655` (dark green)
- NEW badge: `#0E6655`
- UPDATED badge: `#d4ac0d` (gold)
- EXPANDED badge: `#2e86c1` (blue)
- Background: `#e8f6f3`

## Customization Notes
- Swap the ACC/AHA badge for your relevant guideline organization (ESC, NCCN, etc.)
- Add or remove change cards as needed
- Recommendation tiers can be adapted to other classification systems
- Ensure guideline references are accurate and up to date
