# 48 - Seasonal Vaccination Reminder

## Description
Seasonal vaccine availability and ordering reminder template for HCPs. Features a blue color scheme with vaccine inventory cards showing stock status badges, storage requirements, and patient population recommendations.

## LSC Features Used
- **LSC_DropDownInput** - Vaccine type selector (Quad Flu, High-Dose, RSV, etc.)
- **LSC_DateTimeInput** - Preferred order/delivery date selector
- **LSC_RichTextInput** - Additional personalized notes for the practice

## Template Features
- Seasonal header with year/season designation
- Vaccine availability cards with colored stock status badges (In Stock/Limited Supply)
- Left-border color coding for availability status (green/amber)
- Storage requirements box with temperature and handling details
- Two-column patient population priority boxes
- Ordering information with date selector
- Clean, inventory-style layout

## Customization Points
- Update season year and vaccine product names
- Modify availability status badges per current supply
- Edit storage requirements for specific vaccines
- Update patient population recommendations per ACIP
- Change ordering portal and distributor information
- Customize compliance codes in footer

## Primary Color
`#1A5276` (Blue)
