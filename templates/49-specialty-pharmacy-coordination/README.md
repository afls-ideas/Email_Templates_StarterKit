# 49 - Specialty Pharmacy Coordination

## Description
Template for coordinating specialty pharmacy services with HCPs. Features a partner pharmacy list, referral process flow steps, and tracking information in a medium teal color scheme.

## LSC Features Used
- **LSC_DropDownInput** - Preferred specialty pharmacy selector
- **LSC_RichTextInput** - Patient-specific coordination notes

## Template Features
- Specialty pharmacy partner table with phone numbers and turnaround times
- Four-step referral process flow with gradient teal step bars
- Numbered step indicators (01-04) with progressive color scheme
- Tracking and status update callout box
- Clean table layout for pharmacy partner comparison
- Patient access support contact in footer

## Customization Points
- Update specialty pharmacy partner names and phone numbers
- Modify turnaround time estimates per actual agreements
- Edit referral process steps for your specific workflow
- Replace product name (RheumaFlex) and access program details
- Update tracking portal URL and status notification details
- Customize patient assistance program information

## Primary Color
`#117864` (Medium Teal)
