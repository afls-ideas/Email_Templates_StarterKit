# 42 - Disease State Education

## Description
Educational email template focused on disease state awareness for HCPs. Features a dark teal color scheme with prevalence statistics highlight boxes and a structured treatment landscape overview.

## LSC Features Used
- **LSC_EmailFragments** - Detailed disease state information sections
- **LSC_PresentationLinks** - Links to educational presentations
- **LSC_RichTextInput** - Personalized relevance-to-practice note

## Template Features
- Top accent bar with teal branding
- Three-column prevalence statistics boxes with bold numbers
- Treatment landscape with color-coded bullet categories
- Unmet needs callout box with left border accent
- Additional resources section for fragments and presentations
- Clean, editorial-style layout

## Customization Points
- Replace disease state (RA) with target indication
- Update prevalence statistics in highlight boxes
- Modify treatment landscape categories and descriptions
- Edit unmet needs section for specific therapeutic area
- Customize product name and compliance codes in footer

## Primary Color
`#0E6251` (Dark Teal)
