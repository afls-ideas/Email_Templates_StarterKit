# 28 - Webinar Invitation

## Description
Invitation to an educational webinar with speaker bio, agenda, CME credit mention, and registration CTA. Features a clean, academic design with prominent date/time display.

## LSC Features Used
- **LSC_DateTimeInput** (`webinarDate`) - Webinar date and time display in the header bar

## Dynamic Fields
- `{{recipient.lastname}}` - HCP last name
- `{{sender.firstname}}` / `{{sender.lastname}}` - Sales rep name
- `{{webinarDate}}` - Webinar date and time

## Design
- **Primary Color:** #0B5345 (forest teal)
- **Layout:** CME badge top-right, split date/time bar, speaker card with avatar, square-bullet agenda list, inverted CTA block (white button on dark background)
- **Tone:** Educational, professional, academic

## Customization Notes
- Update webinar topic, speaker details, and institution
- Modify agenda bullet points for actual content
- Adjust CME credit amount and accreditation details
- Replace Pulmonex and BREATHE trial names with your product/trial
- Update duration and Q&A details as needed
