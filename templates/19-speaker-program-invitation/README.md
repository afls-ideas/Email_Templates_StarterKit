# 19 - Speaker Program Invitation

## Description
An elegant invitation to a speaker program and dinner event featuring a KOL presenting on ImmunoGuard in rheumatoid arthritis. Formal, event-style design with speaker bio and venue details.

## LSC Features Used
- **LSC_DateTimeInput** (`event_date`) - Event date and time
- **LSC_EmailFragments** (`speaker_details`) - Insert additional speaker bio, publications, or credentials

## Template Sections
- Elegant centered header with "Cordially Invited" styling and decorative lines
- Program title and topic in italic
- Featured speaker bio card with avatar placeholder
- Two-column event details (date/time + venue)
- Prominent RSVP CTA with deadline note
- Compliance footer noting sponsorship and speaker payment

## Color Palette
- Primary: `#7D3C98` (rich purple)
- Gradient: `#7D3C98` to `#5B2C6F`
- Accent: `#d7bde2`
- Background: `#f4ecf7`

## Customization Notes
- Speaker bio section should be populated with your actual KOL details
- Venue name and address are placeholders
- RSVP deadline uses {{rsvp.deadline}} token - set this in your campaign
- Cocktail/dinner times can be adjusted
- Ensure speaker disclosure (paid consultant) complies with transparency requirements
