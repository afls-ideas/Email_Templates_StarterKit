# 46 - Prescribing Information Update

## Description
Template for notifying HCPs about updates to a product's Prescribing Information (label). Features a structured change summary table comparing previous and updated language, with a dark charcoal professional design.

## LSC Features Used
- **LSC_DropDownInput** - PI section of interest selector
- **LSC_EmailFragments** - Detailed PI section content

## Template Features
- Dark charcoal header with product name badge
- Effective date, PI version, and NDA info bar
- Four-row change summary table (Section, Previous, Updated columns)
- Green-highlighted cells for updated language
- Key highlights callout box with green accent
- Full PI link CTA button
- Clean, regulatory-document-inspired layout

## Customization Points
- Update product name and NDA number
- Modify change summary table rows for actual PI changes
- Replace effective date and PI version number
- Edit key highlights section
- Update full PI URL link
- Customize section options in the dropdown

## Primary Color
`#2C3E50` (Dark Charcoal) with `#3498db` (Blue) and `#27ae60` (Green) accents
