# 44 - Medical Information Resource

## Description
Template for sharing medical information and MSL resources with HCPs. Features a resource catalog layout with categorized information sections and an MSL contact card, using a steel blue color scheme.

## LSC Features Used
- **LSC_DropDownInput** - Information category selector (Clinical Data, Dosing, Safety, etc.)
- **LSC_RichTextInput** - Free-text field for specific medical information questions
- **LSC_EmailFragments** - Detailed medical information content sections

## Template Features
- Branded header with "MI" icon badge
- Four-category resource catalog with color-coded headers
- Structured request form with dropdown and free-text input
- MSL contact card with phone and credentials
- Medical information center hotline reference
- Professional card-style layout for each resource category

## Customization Points
- Update resource categories and descriptions for your product
- Replace product name (ImmunoGuard) and trial names
- Modify MSL credentials and contact information
- Change medical information center phone number
- Update category options in the dropdown selector
- Customize compliance codes in footer

## Primary Color
`#1B4F72` (Steel Blue)
