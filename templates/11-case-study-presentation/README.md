# 11 - Case Study Presentation

## Description
Presents an anonymized clinical case study to HCPs, featuring Neuralief for treatment-resistant neuropathic pain. Designed to facilitate evidence-based discussion between reps and physicians.

## LSC Features Used
- **LSC_RichTextInput** (`rep_commentary`) - Allows the rep to add personalized commentary on the case
- **LSC_EmailFragments** (`additional_cases`) - Insert additional case studies as needed

## Template Sections
- Header with "Clinical Case Study" badge
- Anonymized patient profile (demographics, diagnosis, prior treatments, comorbidities)
- Treatment timeline with visual week-by-week progression
- Key outcomes metrics (pain reduction, AEs, duration)
- Rep commentary block
- Additional cases fragment slot
- CTA button for scheduling a discussion

## Color Palette
- Primary: `#884EA0` (violet)
- Gradient: `#884EA0` to `#6C3483`
- Background: `#f0ecf3`
- Accent: `#d7bce8`

## Customization Notes
- Replace Neuralief product details with your approved product
- Update patient profile fields to match your therapeutic area
- Timeline milestones can be added or removed as needed
- Outcomes metrics are flexible - swap in relevant KPIs
- All clinical claims must be reviewed by medical/legal/regulatory
