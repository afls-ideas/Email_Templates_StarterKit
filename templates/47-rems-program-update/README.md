# 47 - REMS Program Update

## Description
Template for communicating REMS (Risk Evaluation and Mitigation Strategy) program updates and requirements to HCPs. Uses a dark burgundy color scheme with a structured, compliance-focused layout.

## LSC Features Used
- **LSC_RichTextInput** - Practice-specific REMS implementation questions
- **LSC_EmailFragments** - Detailed REMS program information sections

## Template Features
- "REMS Required" badge in header
- Compliance deadline prominently displayed in sub-header bar
- Three numbered REMS requirements with square badges
- Patient monitoring requirements bullet list
- Reporting obligations section
- REMS Portal CTA button
- REMS support contact information in footer

## Customization Points
- Replace product name (NeuroCalm) with target product
- Update compliance deadline date
- Modify REMS requirements for specific program
- Edit patient monitoring requirements list
- Change reporting obligations and timelines
- Update REMS portal URL and support contact info

## Primary Color
`#7B241C` (Dark Burgundy)
