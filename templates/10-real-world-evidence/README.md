# 10 - Real-World Evidence

## Description
Presents real-world data for Dermacel in atopic dermatitis from a large registry study. Features patient population demographics, key outcomes with visual stat blocks, and data source context.

## Primary Color
Steel Blue (#1F618D)

## LSC Features Used
- **LSC_RichTextInput** (`personalContext`) - Personalized context area for the rep to add relevance to the HCP's practice
- **Mustache Variables** - `{{recipient.name}}`, `{{sender.firstname}}`, `{{sender.lastname}}`, `{{sender.title}}`, `{{sender.company}}`

## Layout Features
- Header with accent underline bar
- Data source callout box with left-border accent
- Four-column patient population stat cards
- 2x2 outcomes grid with dark header row and light detail row
- RWE disclaimer note box

## Customization Notes
- Update registry name, patient count, and follow-up duration in the data source box
- Modify the four patient population stat cards with your study demographics
- Replace outcomes data (EASI-75, persistence, NRS, DLQI) with your product's RWE results
- Adjust the disclaimer language per your medical-legal review requirements
