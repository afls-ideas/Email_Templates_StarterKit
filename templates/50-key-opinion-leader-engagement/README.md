# 50 - Key Opinion Leader Engagement

## Description
Premium outreach template for engaging Key Opinion Leaders (KOLs) for advisory boards and scientific collaborations. Features a dark, exclusive design with gold accents and serif typography for a distinguished feel.

## LSC Features Used
- **LSC_RichTextInput** - Personalized pitch explaining why the KOL's expertise is valued
- **LSC_DateTimeInput** - Proposed introductory meeting date/time

## Template Features
- Dark background (near-black) with gold accent design throughout
- Gold diamond ornamental header elements
- Serif font (Georgia) for premium editorial feel
- Structured engagement details card with gold header
- Honorarium range and time commitment displayed
- Past collaboration reference section
- "Express Interest" gold CTA button
- Confidentiality notice
- PhRMA Code and Sunshine Act compliance reference

## Customization Points
- Update engagement type (Advisory Board, Speaker Bureau, etc.)
- Modify honorarium range and time commitment
- Replace program name and therapeutic area
- Edit past collaboration references for specific KOL
- Update sender title for appropriate seniority level
- Customize compliance language for your organization

## Primary Color
`#1C2833` (Near Black) with `#D4AC0D` (Gold) accent
