# 16 - Lunch & Learn Invitation

## Description
Invites an HCP and their office staff to a catered lunch and learn session covering COPD triple therapy with Pulmonex. Warm, inviting design with event details in a grid layout.

## LSC Features Used
- **LSC_DateTimeInput** (`event_datetime`) - Event date and time selector
- **LSC_DropDownInput** (`dietary_preference`) - Dietary preference for catering (Vegetarian, Vegan, Gluten-Free, etc.)

## Template Sections
- Centered hero header with food icon and event title
- 2x2 event details grid (date, location, duration, catering)
- Topic overview with bullet list
- RSVP section with dietary preference dropdown and CTA button
- Compliance-aware footer

## Color Palette
- Primary: `#D35400` (warm orange)
- Accent: `#fad7a0` (light gold)
- Card backgrounds: `#fef5e7`
- Background: `#fdf2e9`

## Customization Notes
- Update the topic and product name for your therapeutic area
- Dietary preference options can be modified
- Catering vendor name is a placeholder - update as needed
- Duration and location fields are editable
- Ensure meal value complies with local sunshine act / transparency requirements
