# 41 - CME Program Invitation

## Description
Email template for inviting Healthcare Professionals to a CME-accredited continuing medical education program. Features a professional midnight blue design with gold accent for the CME credits badge.

## LSC Features Used
- **LSC_DateTimeInput** - Date and time selector for the program schedule
- **LSC_DropDownInput** - Session preference selection (Morning/Afternoon/Evening)

## Template Features
- CME credits badge with circular gold design (3.5 credits)
- ACCME accreditation information
- Four numbered learning objectives
- Featured speaker profile with avatar placeholder
- Program details with structured layout
- Registration CTA button
- Professional footer with compliance codes

## Customization Points
- Update CME credit count in the gold badge
- Replace speaker name, title, and credentials
- Modify learning objectives for your specific program
- Adjust session time options in the dropdown
- Update accreditation and compliance text in footer
- Change registration deadline date

## Primary Color
`#154360` (Midnight Blue) with `#d4ac0d` (Gold) accent
