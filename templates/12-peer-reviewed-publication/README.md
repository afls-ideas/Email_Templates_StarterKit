# 12 - Peer-Reviewed Publication

## Description
Highlights a newly published peer-reviewed journal article (SHIELD-RA Phase III trial for ImmunoGuard). Uses an academic, journal-inspired design with serif typography for a scholarly feel.

## LSC Features Used
- **LSC_PresentationLinks** (`full_paper`) - Links to the full publication or approved reprint

## Template Sections
- Top bar with publication alert label and date
- Journal citation block (title, journal name, volume, DOI)
- Key findings with left-bordered highlight cards
- Author information panel (dark background)
- Full paper access CTA
- Rep signature and compliance footer

## Color Palette
- Primary: `#2C3E50` (charcoal)
- Secondary: `#5d6d7e`, `#85929e`, `#aeb6bf` (graduated grays)
- Background: `#eaecee`

## Customization Notes
- Replace journal citation details with your actual publication
- Key findings cards can be added or removed (maintain border-left color gradient)
- Author section supports any number of authors
- Uses Georgia/serif font for academic feel - change to Arial for a more modern look
- Ensure all publication claims are MLR-approved before sending
