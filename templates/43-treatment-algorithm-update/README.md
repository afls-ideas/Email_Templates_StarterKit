# 43 - Treatment Algorithm Update

## Description
Template for communicating treatment algorithm changes and guideline updates to HCPs. Features a numbered step visualization with highlighted changes and an imperial purple color scheme.

## LSC Features Used
- **LSC_DropDownInput** - Patient population/type selector (EGFR+, ALK+, PD-L1, etc.)
- **LSC_EmailFragments** - Detailed algorithm information sections
- **LSC_PresentationLinks** - Links to algorithm presentations and visual aids

## Template Features
- Numbered step-based algorithm visualization with circle indicators
- Vertical connector lines between steps
- Yellow-highlighted "NEW CHANGE" badges on updated steps
- Gradient purple header with guideline version badge
- Guideline source citation box
- Five-step treatment flow from testing to monitoring

## Customization Points
- Replace NSCLC with target indication
- Update algorithm steps for specific treatment pathway
- Modify patient population options in dropdown
- Change highlighted steps to reflect actual guideline updates
- Update guideline source and version number
- Customize compliance codes in footer

## Primary Color
`#4A235A` (Imperial Purple) with `#f0b429` (Gold) for change highlights
