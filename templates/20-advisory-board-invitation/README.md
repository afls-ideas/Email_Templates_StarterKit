# 20 - Advisory Board Invitation

## Description
A premium, exclusive invitation to join the Neuralief Advisory Board. Dark theme with gold accents creates a high-end, prestigious feel appropriate for senior KOL engagement.

## LSC Features Used
- **LSC_DateTimeInput** (`board_date`) - Advisory board meeting date
- **LSC_RichTextInput** (`invite_reason`) - Personalized explanation of why this HCP's expertise is valued

## Template Sections
- Gold gradient top border with "Exclusive Invitation" badge
- Centered title with gold accent subtitle
- Personalized invite reason block (gold left border)
- Detailed event info (date, location, duration, honorarium)
- Full-day agenda preview with timed sessions
- Gold "Accept Invitation" CTA with secondary decline option
- Compliance footer noting honorarium and legal requirements

## Color Palette
- Primary: `#1C2833` (near-black background)
- Accent: `#C9A227` (gold)
- Text: `#ecf0f1` (light), `#85929e` (muted)
- Card background: `#212f3d`

## Customization Notes
- This is a dark-theme template - ensure your email client supports dark backgrounds
- Honorarium language should be reviewed by legal/compliance
- Venue and agenda are placeholders - update for your actual event
- The personalized invite reason (LSC_RichTextInput) is critical for KOL engagement
- Travel/accommodation details can be added to the event details section
