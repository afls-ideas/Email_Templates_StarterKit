# 45 - Safety Update Notification

## Description
Urgent safety communication template for notifying HCPs of important drug safety updates. Features prominent red/warning styling with checklist-style action items and adverse event reporting information.

## LSC Features Used
- **LSC_RichTextInput** - Practice-specific guidance notes from the rep

## Template Features
- Urgent red top banner with warning symbols
- Dark red gradient header with alert classification
- Red-bordered alert box requiring provider attention
- Six-item healthcare provider action checklist with checkboxes
- Updated warnings section with left red border accent
- Adverse event reporting contact box (FDA MedWatch + company)
- Serious, professional tone throughout

## Customization Points
- Update drug name and safety signal details
- Modify action items checklist for specific safety concern
- Replace adverse event reporting contact information
- Update date of issue and reference number
- Edit warning and precaution text
- Customize LFT thresholds and monitoring requirements

## Primary Color
`#C0392B` (Alert Red) with `#922b21` (Dark Red) accents
