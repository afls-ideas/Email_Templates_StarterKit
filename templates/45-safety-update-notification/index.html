Dear {{recipient.name}},
Safety Signal Summary
Post-marketing surveillance and ongoing clinical trial monitoring have identified a new safety signal associated with OncoVex (oncovelimab) treatment. An increased incidence of grade 3-4 hepatotoxicity has been observed, particularly in patients receiving concurrent hepatotoxic medications.
Updated Warnings & Precautions
NEW WARNING: Severe hepatotoxicity, including cases of hepatic failure, has been reported. Liver function tests (ALT, AST, bilirubin) should be monitored prior to each treatment cycle.

UPDATED PRECAUTION: Concurrent use with other hepatotoxic agents should be avoided when possible. If co-administration is necessary, more frequent LFT monitoring (every 2 weeks) is recommended.
Required Healthcare Provider Actions
 
 Review all current patients receiving OncoVex for hepatotoxicity risk factors
 
 Implement baseline and pre-cycle LFT monitoring for all patients
 
 Review concomitant medications for potential hepatotoxic interactions
 
 Withhold treatment if ALT/AST > 5x ULN; discontinue if ALT/AST > 10x ULN
 
 Educate patients on signs/symptoms of hepatotoxicity to report immediately
 
 Report adverse events to FDA MedWatch and to OncoVex Pharmacovigilance
